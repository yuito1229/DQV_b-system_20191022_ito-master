# ドラゴンクエストVの戦闘システム
## 動作するIDE
・Windows版のVisualStudio
## 操作方法
・コマンドの選択は数字で行う。上から順に1〜9、0を割り当てている。コマンド数の最大値は10である。<br>
・Enterを押すとコマンド選択を確定する。なお、コマンドを戻ることはできないので注意が必要。
## じゅもんの中のコマンドとどうぐの中のコマンド
